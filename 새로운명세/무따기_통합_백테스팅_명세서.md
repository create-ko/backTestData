# 무따기 매매 시스템 — 통합 백테스팅 명세서

> 두 문서(『더블비 백테스팅 규칙』 + 『프로 백테스팅 명세서』)를 하나로 합친 통합본.
> 중복 항목은 병합했고, **두 문서가 서로 다르게 적은 부분은 §11 충돌·미해결 표**에 모았다.
> 표기 규칙: 강의 원문 근거 = 표기 없음 / 임의 해석 = `※` / 원문 미명시·튜닝 필요 = `[가정]`

## 0. 공통 전제

| 항목 | 기준 |
|---|---|
| 메인 타임프레임 | 1시간봉 (물리면 4시간봉까지 확인) |
| 보조지표 | 볼린저밴드 — 진입 신호 **4/4값**, 추세·익절 판단 **20/2값** (가끔 4/4 → 3/3으로 변경해 봄) |
| 진입 신호 핵심 | **더블비(원비/더블바텀)** — 더블비가 아니면 변곡의 시작이 아니므로 진입하지 않음 |
| 방향 판단 | **더캔이지추격깨패** (§2) — 매수우위/매도우위 확정 후 그 방향으로만 진입 |
| 봉 마감 조건 | 해당 방향 **몸통 마감**이 best, 꼬리 마감은 so-so, 꼬리 달고 말아올리면 무효 |
| 손절 기준 시점 | **완전히 이탈한 1시간봉**을 보고 손절 (봉 진행 중 이탈은 무시) |
| 금지 시간대 | 지표 발표 시간대(예: 새벽 3시), 미장 시작(23시) 직후 신규 진입 회피. 지표 때 꼭 하려면 1%만 |
| 비중 원칙 | 평소 1%, 비중베팅 조건 충족 시에만 키움. 한 번 익절하면 바로 이어서 거래하지 않음 |

나스닥 = 1 / 골드 = 100 / 오일 = 100 / 비트 = 1

## 1. 지표·신호 정의

| 지표 | 정의 | 백테스트 근사 |
|---|---|---|
| 20 SMA | 종가 20기간 단순이동평균 | — |
| 120 SMA | 종가 120기간 단순이동평균 | — |
| 볼린저밴드(추세·익절) | 20 SMA ± 2σ (20/2) | 표준 세팅 |
| 볼린저밴드(진입신호) | 4/4값 (가끔 3/3) | — |
| 원비 | 가격이 볼밴 상·하단에 닿음 + 추가 조건. "20/2만 맞으면 원비라 안 함" | `[가정]` 밴드 터치로 근사 |
| 수축원비 | 몸통으로 눌러주는(정확히 찍는) 원비 | 몸통 마감 조건으로 근사 |
| 더블비 | 원비가 연속·중복 발생(밴드 재터치) | `[가정]` 밴드 2회 터치로 근사 |
| 쌍비 | 반대 방향에서 나오는 더블비형 신호(역방향 신호) | — |

> ⚠️ 원비·더블비·쌍비의 정밀 정의는 강의에 **그림으로만** 제시되어 텍스트 정의가 빈칸이다.
> 백테스트에서는 **밴드 터치 + 봉마감**으로 근사하되, 이 정의 자체를 파라미터로 두고 검증할 것.

## 2. 방향 판단 — 더캔이지추격깨패

진입 전, **내가 진입한 프레임 자체의 우위**로 매수우위/매도우위를 결정한다(상위 프레임 아님).
**최소 3가지 이상 충족 시 진입.**

| 약자 | 항목 | 매수우위 | 매도우위 |
|---|---|---|---|
| 더 | 더블비/원비(수축원비) | 매수 방향 더블비 | 매도 방향 더블비 |
| 캔 | 캔들 | 망치형, 밑꼬리 없는 양봉 | 역망치형, 상승 중 도지, 윗꼬리 없는 음봉 |
| 이 | 이평(20/120) | 골든크로스 정렬·이평 올라탐 | 데드크로스 정렬·이평 하향돌파 (※ 방향 자체는 크게 중요시 안 함) |
| 지 | 지지/매물대 | 가까운 매물대가 **아래** ('선' 아닌 매물대로 판단) | 가까운 매물대가 **위** |
| 추 | 추세선 | 상방 추세선 (방향만 식별, 타점 아님) | 50% 되돌림 자리·하방 추세선 |
| 격 | 이격도(0.5 단위) | 과매도권·다이버전스 | 과열권·다이버전스 |
| 깨 | 그날 시가 | 시가/고가/저가 상향 돌파 | 시가/고가/저가 하향 이탈 |
| 패 | 차트패턴 | 더블바텀, 역헤드앤숄더 | 더블탑(M자), 헤드앤숄더 |

> 핵심: **모든 것은 더블비로 귀결**되며, 나머지는 그 더블비 방향이 그날 일봉 방향과
> 부합하는지를 검증하는 보조 근거다.

> `[가정]` 각 요소의 가중치는 미명시. 백테스트는 **"20/120 크로스 방향"을 1차 필터**로,
> 나머지를 보조 confirmation으로 두는 단순 룰부터 시작 권장. 6요소 다수결 점수화도 파라미터화 가능.

## 3. KTR (김직선 TR) — 변동성 단위

```
KTR = (장 시작 직후 첫 1개 봉의 고가 - 저가)   # 그 봉의 True Range
```

**규칙**
- 측정 봉: **장 시작 직후 첫 1개 봉** (직선님은 주로 10분봉 사용)
- **아시아장**은 예외적으로 **첫 2개 봉을 묶어서** 측정
- **지표 발표 직후 첫 1개 봉**도 KTR 후보. 지표봉이 장 시작봉보다 크면 **지표봉 채택**
- 값은 **0.5단위** 반올림
- **아시아 / 유로 / 미장** 세션별 각각 계산
- KTR 확정(봉마감) 전까지는 **아무 행동도 하지 않음**

**입력 파라미터**: KTR 측정 TF(1/2/5/10분, 기본 10분), 세션 시작 시각(거래소 기준)

### 3.1 랏수 계산
```
가능랏수 = (증거금 × 리스크%) / (N × KTR × $)
  증거금  : 1,000 ~ 100,000 불
  리스크% : 1 ~ 20%
  N       : 손절/익절 KTR 배수 (§5 참조)
  $       : 나스닥1 / 골드100 / 오일100 / 비트1
예) 증거금 1,000 · 리스크 10% · KTR 30p · N 3 → 약 1.11랏 (나스닥)

랏수는 [리스크%↑ / 변동성↓ / KTR↓] 일수록 커짐
```

## 4. 진입 규칙

### 4.0 거래 마인드 두 가지
| 구분 | 데이 트레이딩(스윙) 마인드 | 트리거(비중배팅) 마인드 |
|---|---|---|
| 성격 | 미리 설계된 시나리오 기반 지정가 | 즉시 포착·대응 |
| 진입 | KTR 배수 자리에 지정가 사전 세팅 | 수축원비 + 봉마감 확인 후 진입 |
| 손절 | 3.5 / 5.5 KTR | 1 ~ 1.5 KTR (2 이상 금지) |
| 비중 | 정률 계산 랏수 | 계산 랏수의 4~10배 |
| 보유 | 길면 하루 | 길어야 1시간 이내 |

> 하루의 시작은 **무조건 데이 트레이딩 마인드**. 미리 진입돼 있지 않으면 트리거 진입도 뇌동매매다.
> 트리거 마인드는 충분한 경험 전엔 사용 금지.

### 4.1 핵심 진입 — 5분봉 트레이딩 Tip (가장 명확, 백테스트 1순위)
```
조건1: 5분봉이 20 SMA 돌파
조건2: 20·120 SMA 사이에서 골든크로스(매수)/데드크로스(매도) 발생
조건3: 위 발생 후 "첫 번째 원비/더블비" 봉마감
→ 크로스 방향 진입
추가 타점: 추세 진행 중 되돌려 120 SMA에 처음 닿는 자리 재진입 가능
```

### 4.2 전략 A — 기본더블비 구간매매
이평 방향과 **같은 방향**으로 볼린저 **반대편**을 찌르는 더블비 진입(정통 구간매매).
- 이평 **상방** + 볼린저 **하단** 찌름 → 매수
- 이평 **하방** + 볼린저 **상단** 찌름 → 매도
- 진입과 동시에 추가진입·손절·익절 시나리오를 모두 세팅한다.

| # | 진입 조건 |
|---|---|
| A-1 | 더캔이지추격깨패 **3개 이상** 충족, 방향 확정 |
| A-2 | 해당 방향 **기본더블비** + **수축원비**(몸통 눌러줌) |
| A-3 | 해당 방향 봉 마감(몸통 우선) |
| A-4 | (선택) 1H 수축원비+봉마감 확인 후 5분봉 추가 근거로 세부 타점 |

### 4.3 전략 B — 돌파더블비 매매
매물대를 돌파하는 방향의 더블비. 추세 추격 성격이라 손익비가 나쁘다.
→ 강의 최강조: **"돌파더블비는 매물대가 없어야만 한다."**

| # | 진입 조건 (A보다 빡빡) |
|---|---|
| B-1 | 진행 방향 **앞쪽에 매물대 0** — 가장 중요한 필터 |
| B-2 | 매물대를 **완전히 이탈/돌파한 봉** 확인 (진행 중 진입 금지) |
| B-3 | 돌파 방향 더블비 + 몸통 마감 (말아올림 무효) |
| B-4 | **SL 필수** |
| B-5 | 비중 적게(1%). 한 번은 못 살아나올 수 있음 전제 |

**절대 금지 (백테스트 제외 조건)**
- **"위에서 잡지 않는다"** — 밑에서 잡아 위로 가는 구조. 매물대 위 돌파 가능성 보여도 진입 금지(돌파 아니면 손익비 붕괴)
- 거래량 터진 상태 진입 = 사실상 돌파매매 → 되돌림 데미지 누적, 지양
- 매물대 두꺼운 구간은 돌파더블비 대신 **기본더블비(전략 A)** 로 접근 (※ 같은 자리도 매물대 유무로 A/B가 갈림)

### 4.4 1구간 회피 필터 (조건부 토글)
```
IF 진행 방향 앞에 반대방향 매물대(더블탑/바텀·주요 고저점) 존재  OR  횡보장
THEN 1구간 진입 생략, 2~3구간부터 진입 (깊은 조정 대비)
  예외: 더블탑/바텀 + 정확히 몸통으로 찍는 수축원비 등장 시 1구간 허용
```
- RSI 3선 체크 필수

### 4.5 원웨이 추세 추종
```
IF |가격 이동| >= 5 KTR (= 원웨이 정의, 탐색범위 4~6 KTR)
THEN 그 방향으로만 구간매매, 반대방향 신호는 눌림으로 보고 무시
```

### 4.6 진입 제약 (공통)
- 반대방향 타점 등장 시 → **익절 O, 신규 진입 X**
- 연속 역방향 진입 **금지** (위아래 다 먹지 않음)

## 5. 추가진입 · 손절 · 익절

### 5.1 추가진입 (구간 분할)
| 프레임 | 한계 | 비고 |
|---|---|---|
| 시간봉 | 3 KTR (평균) | 1포지션부터 분할 |
| 분봉 | 5 KTR (평균) | |

- 구간 매매(B 마인드): 직전 더블비~무빙 멈춘 지점까지 레인지 측정 → KTR로 분할 → 각 구간 경계 지정가 사전 세팅(물타기 아님)
- 물려 5포지션 부근까지 오면 마지막 포지션을 크게 잡아 전체 정리도 가능 (이 경우 SL = 마지막 포지션 위 1KTR)
- 돌파더블비(B 전략) 추가구간도 **매물대 비어 있어야** 함
- ※ 백테스트는 "균등 분할" vs "후반 가중" 두 케이스 분리 검증

### 5.2 손절
| 마인드/프레임 | 손절 |
|---|---|
| 데이(시간봉) | **3.5 KTR** (PRO 본문 3.3 표기 → 두 값 모두 테스트) |
| 데이(분봉) | **5.5 KTR** |
| 트리거(비중배팅) | **1 ~ 1.5 KTR** (2 이상 절대 금지) |
| 5분봉 Tip | 고저점 자체 아닌 그 **부근 KTR 정수배**(예: 3KTR)에 설치 |

- 손절가는 항상 KTR 배수로 표현(소수배 허용)
- 전고/전저·장기이평·매물대를 손절선 근처에서 같이 확인
- 손절 자리에선 강력한 반등이 1회 이상 출현하는 경향 → **손절 직후 재진입(수축원비+봉마감)** 케이스 별도 기록

### 5.3 익절
```
기본: 손절폭의 KTR 정수배(소수배) 지점에서 익절

방법: (1) 전 포지션 동시 청산
      (2) 포지션별 KTR 배수 구간에서 분할 청산 + 1개는 남겨 트레일링

트레일링 스탑: 0.5 KTR 또는 1 KTR 추적
이익 보존:    25% / 50% / 75% 단계 트레일링
1차 목표가:   더 큰 TF(2·4시간·일봉)의 수축원비 지점
강제 청산:    진행 중 장기이평이 원비를 찍거나 더블탑/바텀 출현 시 전체 정리
```
- 구체적 익절 지점: 4볼밴 닫히는 구간 / 이평·다음 원비 / 반대 타점 등장 / 매물대 박스 직전
- 단기 익절 관행: 골드 2~3p, 나스닥 10~20p씩 자주 (비중으로 보완)
- 돌파더블비(B)는 **짧게** + 완전 돌파 확인 후 추격(빈 공간에서 빠르게), 비중 커도 길게 홀딩 X
- 본인 익절 TR 기준은 통계로 도출 (백테스트 핵심 산출물)

## 6. 비중(랏수) 규칙
```
구간별 차등 예시: 1구간 1% / 2구간 3% / 3구간 5%  (증거금 대비 리스크 %)
랏수 ∝ 리스크%↑ · 변동성↓ · KTR↓
```

## 7. 기본더블비 vs 돌파더블비 — 분류 기준

| 구분 | 기본더블비 (A) | 돌파더블비 (B) |
|---|---|---|
| 진행 방향 매물대 | 있어도 됨(아래/뒤 두꺼우면 유리) | **없어야 함** |
| 캔들 힘·모양 | 반대편 찌르고 되돌리는 더블비 | 매물대 뚫는 강한 돌파봉 |
| 손익비 | 양호 | 나쁨 (1%·SL 필수) |
| 비중 | 조건 충족 시 키움 | 적게 |
| 익절 | TR 통계/트레일링 | 짧게(나스닥 ~10p, 골드 ~1p) + 완전 돌파 후 |

> 구분 판정 순서: **캔들패턴 → 이평 → 지지저항/매물대 → 목표가/손절가 → 추세선 → 이격도 다이버전스 + 차트패턴**

## 12. 해석 시 핵심 주의사항
- **"나무(더블비)만 보고 들어가라"** — 차트 전체 다 보면 못 잡음. 백테스트도 **진입 시점까지의 정보만으로** 판단했는지 검증(미래참조 편향 주의)
- 승률 높은 전략(기본더블비)이 장기 생존에 유리. 손익비만 높고 승률 낮은 전략은 승률 무너지는 순간 청산
- **손절 못 하면 9번 이겨도 1번에 죽는다** — 손절 미실행 시나리오의 파괴력을 반드시 시뮬레이션
- **"디테일은 비법이 아니다 / 스스로 만들지 않은 규칙은 실수가 나온다"** — `[가정]` 파라미터는 직접 grid search로 튜닝할 것
